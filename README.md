
# 项目简介
网络爬虫项目

---

## 当前功能
### 脚本：`web_scraper_requests_bs4.py`
- **功能**：
  1. 从 API 获取分页数据。
  2. 解析网页内容，提取特定标签信息。
  3. 整合数据并输出为 JSON 格式。
- **依赖**：
  - `requests`
  - `beautifulsoup4`

### 运行方式
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
2. 运行脚本：
   ```bash
   python web_scraper_requests_bs4.py
